RULES

Gameplay:
1. No leaving the normal boundaries of a Map
2. No going onto Invisible Ledges.
3. No using the Grenade Launcher.
4. Bandolier is the only perk allowed in the Perk 1 slot. No using C4, Special Grenades x3,
RPG-7, Claymore, Frag x3, and Bomb Squad. The Grip attachment is permitted.
5. Martyrdom and Last Stand are NOT allowed in the Perk 3 slot.
6. LMGs are not allowed
7. Players may not look at an opposing Player’s screen.
8. If a Game should tie, the Game must be replayed.
9. Quitting from a Round/Game, before it has ended, without permission, will result in
disqualification.
10. Any cheating will lead to disqualification from the tournament
Game Settings:
1. Game Type : Search and Destroy
2. Victory condition : First team to win 8 rounds
3. Each round shall last 3 minutes
4. Each game will consist of 4 matches between competing teams. One match each in Backlot,
Crash, Pipeline and Ambush
5. In case of tie, individual scores in each map would be used to determine victory
6. Spectate = Team Only
7. Friendly Fire = On Kill Cam = Off
8. Air Support = On

Registration - Rs. 50 per head
